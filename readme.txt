User Journey
the user land i nthe home page and click the star btn
the user select the amount of moves that he wants to do (20,30,40,50)
the user select the amout of time between each move is given (3,5,7,10)
the user selects the amounts of Reps (1,2,3,4)
the user selects the amounts of Reps rest time (2,3,4,5)
the user selects the amounts of Set (1,2,3,4)
the user selects the amounts of Set Rest Time (5,10)
the user click the start climbing btn.
